# 🧬 Projet Biometrie

Application Django + React pour la gestion et la recherche biométrique.

## 🚀 Technologies

- **Backend** : Django + MySQL
- **Frontend** : React
- **Base de données** : MySQL

## ⚙️ Installation

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```
